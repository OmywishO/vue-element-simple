# vue-element-simple

## 项目初始化

```
bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

有关工作原理的详细说明，请查看 [VUE](https://cn.vuejs.org/v2/guide/installation.html)

## 项目概述
vue3框架项目搭建，需要 Node.js 版本 >= 12。
